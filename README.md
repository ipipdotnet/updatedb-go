# updatedb-go

···
./updatedb -h  
Usage of ./updatedb:   
      --name string    --name=文件名  
      --path string    --path=要保存的本地路径  
      --token string   --token=TOKEN  
···
> example
````
./updatedb --name=ipip.ipdb.zip --path=/home --token=TOKEN
````
